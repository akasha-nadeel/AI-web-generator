# Audio assets

Drop `ambient-pad.mp3` here to add the music track to the video.

## Where to find a track

- **Pixabay** (free, royalty-free): https://pixabay.com/music/search/ambient%20pad/
  Filter for "Ambient" / 60-90s long. Recommended search terms:
  `ambient pad`, `minimal cinematic`, `tech intro calm`.
- **YouTube Audio Library** (free, attribution sometimes required):
  https://studio.youtube.com/channel/UC/music — search "ambient" / "calm".
- **Artlist / Epidemic Sound** (paid, premium quality) if you want
  Linear / Stripe-tier production.

## Specs to match

- **Vibe:** Held chord with subtle filter sweep. No melody, no rhythm.
- **Length:** ≥75 seconds (video is 72s, give yourself a tail).
- **Mix:** ~ -18 LUFS. The `<AudioTrack>` component plays it at 0.45 gain.
- **File format:** MP3, 44.1 kHz, 192 kbps minimum.

## Saving

Rename the downloaded track to `ambient-pad.mp3` and place it next to this README. Then re-run `npm run render` — the file is auto-detected.
